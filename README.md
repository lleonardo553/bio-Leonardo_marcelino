# 🌐 Leozinho | Bio Links

Um site de links na bio elegante e minimalista, inspirado no Linktree, desenvolvido com HTML, CSS e JavaScript puro.

![Preview](https://screenshot-taker-prod.weichiang.workers.dev/message/019d12b2-b400-7f37-ae92-b5e15779d5a5/og-image.webp)

## ✨ Funcionalidades

- 📱 **Design responsivo** - Funciona perfeitamente em desktop e mobile
- 🎨 **Interface minimalista** - Foco na simplicidade e usabilidade
- ⚡ **Carregamento rápido** - Código otimizado e leve
- 🔗 **Integração com redes sociais** - Centralize todos seus links importantes
- 🖼️ **Preview otimizado** - Meta tags Open Graph e Twitter Cards configuradas

## 🚀 Tecnologias

- **HTML5** - Estrutura semântica
- **CSS3** - Estilização moderna com Flexbox
- **JavaScript** - Interatividade nativa
- **Google Fonts** - Tipografia Inter
